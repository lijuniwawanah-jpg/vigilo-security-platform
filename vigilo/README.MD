# Vigilo Security & Monitoring System

## Description
Vigilo is a security and monitoring platform with location tracking, QR code scanning, and OpenStreetMap integration. Built with PHP, MySQL, and XAMPP.

## Features
- User authentication and OTP verification
- File/document management
- QR code scanning for items
- OpenStreetMap integration for location tracking
- Admin dashboard with stats and activity logs

## Tech Stack
- PHP, MySQL, XAMPP
- HTML, CSS, JavaScript
- OpenStreetMap API
- QR code scanning library

## Setup
1. Install XAMPP and MySQL
2. Copy project files to `htdocs/vigilo`
3. Create database and import tables from `check_tables.php`
4. Update `/config/db.php` with your database credentials
5. Open `http://localhost/vigilo/public/index.php` in browser

## Note
Uploads folder not included in repo. Add your own test data if needed.
